# 📈 Stock Price Prediction using Machine Learning

## Overview

This project predicts the next day's stock closing price using historical stock market data obtained through Yahoo Finance.

## Features

- Data fetched using yfinance API
- Exploratory Data Analysis
- Linear Regression Model
- Random Forest Regressor
- Model Comparison
- Feature Importance Analysis
- Interactive Streamlit Web Application

## Dataset

Historical Apple (AAPL) stock data from Yahoo Finance.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit
- yfinance

## Outputs

- Closing Price Trend
- Actual vs Predicted Prices
- Random Forest Predictions
- Feature Importance Graph
